# Pikeworks Privacy Policies

Public privacy policy pages for Pikeworks apps. Served via GitHub Pages at
https://edm-92.github.io/pikeworks-privacy/

Each `*.html` file is the privacy policy for one app. They are linked from
the Play Store listing of every Pikeworks app and from each app's in-app
Settings → Privacy Policy screen.

This repository is intentionally public — only privacy policy pages and this
README belong here. Internal documentation, server configuration, and any
build tooling lives elsewhere.
